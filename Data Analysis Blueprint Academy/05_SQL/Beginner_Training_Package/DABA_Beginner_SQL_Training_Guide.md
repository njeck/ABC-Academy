# DABA Beginner SQL Training Package

## Package contents

1. **DABA_Beginner_SQL_Student_Workbook.sql**  
   Contains 12 guided practical labs, business analysis challenges, a 50-mark final assessment, and student reflection questions.

2. **DABA_Beginner_SQL_Instructor_Answer_Key.sql**  
   Contains complete reference queries for every practical exercise and assessment question.

## Prerequisites

Run these files first:

1. `ABC_Retail_Phase1_SQLServer.sql`
2. `ABC_Retail_Phase1_SeedData.sql`

Then open the student workbook in Microsoft SQL Server Management Studio.

## Recommended teaching schedule

| Session | Topics |
|---|---|
| Day 1 | SELECT, columns, aliases, TOP |
| Day 2 | WHERE, AND, OR, IN, BETWEEN |
| Day 3 | ORDER BY, DISTINCT, calculated columns |
| Day 4 | COUNT, SUM, AVG, MIN, MAX |
| Day 5 | GROUP BY and HAVING |
| Day 6 | INNER JOIN and LEFT JOIN |
| Day 7 | CASE, date functions, text functions |
| Day 8 | Subqueries and business analysis |
| Day 9 | Revision and guided project |
| Day 10 | Final assessment |

## Assessment structure

- Guided laboratories: 40%
- Business analysis challenge: 20%
- Final SQL assessment: 30%
- Participation and professional practice: 10%

## Learning outcomes

Students should be able to:

- Retrieve and filter relational data.
- Sort and summarize results.
- Calculate business measures.
- Join related tables.
- Categorize records using CASE.
- Work with dates and text.
- Use aggregate functions and grouped analysis.
- Use subqueries.
- Answer realistic business questions with SQL.
- Communicate results clearly.

All data in the ABC Retail Ltd database is fictional and intended only for education.
