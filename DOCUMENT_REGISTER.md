## Control rules

- Instructor answer keys and restricted assessment materials should not be published in a public student repository.
- Each revised controlled document should update its version, date, owner, and change history.
- Superseded documents should be archived and marked as superseded rather than duplicated in active folders.
- Repository paths should be recorded in the Excel master roadmap.
