# Data Analysis Blueprint Academy (DABA)

**Learn. Practice. Analyze. Transform.**

Data Analysis Blueprint Academy is a practical learning ecosystem designed to develop job-ready data professionals through guided lessons, realistic datasets, business simulations, assessments, and portfolio projects.

This repository contains the academy's curriculum, governance documents, training datasets, SQL and Power BI resources, AI-for-analysts materials, assessments, and the integrated **ABC Retail Ltd** enterprise simulation.

## Project status

- Flagship program: Six-Month Diploma in Data Analytics
- Core learning areas: Excel, business statistics, SQL, Power BI, AI for Data Analysts, and capstone delivery
- Enterprise simulation: ABC Retail Ltd
- Data classification: Synthetic educational data only
- Founder: Mbah Dousbel Angum

## Learning pathway

1. Foundations of Data Analytics
2. Excel Fundamentals for Analysts
3. Data Cleaning and Analysis in Excel
4. Business Statistics and Analytical Thinking
5. SQL and Relational Databases
6. Power BI and Business Intelligence
7. AI for Data Analysts and Responsible Analytics
8. Integrated Capstone and Career Readiness

## Getting started

### Students

1. Start with the six-month curriculum under `02_Academics`.
2. Complete the Excel training datasets and practical workbook.
3. Install SQL Server and run the ABC Retail database scripts in the documented order.
4. Complete the SQL student workbook.
5. Build the Power BI model and dashboard using the source workbook and DAX guide.
6. Complete the responsible AI module.
7. Submit and defend the integrated capstone project.

### Instructors

1. Review the curriculum, instructor guides, and assessments.
2. Validate all datasets and answer keys before delivery.
3. Use synthetic data only.
4. Require learners to document formulas, queries, DAX measures, assumptions, and validation checks.
5. Require disclosure and validation of material AI-assisted work.

## ABC Retail Ltd database setup

1. Open Microsoft SQL Server Management Studio.
2. Run `ABC_Retail_Phase1_SQLServer.sql`.
3. Run `ABC_Retail_Phase1_SeedData.sql`.
4. Review the validation summary returned by the seed script.
5. Use the data dictionary and ERD to understand relationships and fields.

## Academic integrity and responsible AI

Learners remain accountable for every submitted result. AI-generated formulas, queries, DAX, findings, or recommendations must be tested, understood, validated, and disclosed where required. Real personal, confidential, or restricted information must not be uploaded into unapproved services.
